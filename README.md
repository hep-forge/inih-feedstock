[![Anaconda Platforms](https://anaconda.org/hep-forge/inih/badges/platforms.svg)](https://anaconda.org/hep-forge/inih)

Feedstock for [inih](https://github.com/benhoyt/inih) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Simple .INI file parser in C, good for embedded systems

## Install

```bash
conda install -c hep-forge -c conda-forge inih
```
