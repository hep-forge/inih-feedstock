# inih-feedstock

[![Build AMD64](https://github.com/hep-forge/inih-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/inih-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/inih-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/inih-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [inih](https://github.com/benhoyt/inih) — part of [hep-forge](https://anaconda.org/hep-forge).

Simple .INI file parser in C, good for embedded systems

## Install

```bash
conda install -c hep-forge -c conda-forge inih
```
